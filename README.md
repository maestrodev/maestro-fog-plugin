# maestro-vsphere-plugin
Maestro plugin providing a "task" to provision instances with vSphere. This
plugin is a Ruby-based deployable that gets delivered as a ZIP file.


Manifest:

* src/vsphere_worker.rb
* manifest.json
* README.md (this file)
